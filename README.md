# 社团管理系统 java800

## 项目概述

社团管理系统是一个基于Web的网站应用，采用前后端分离的架构设计，主要服务于校园社团的管理工作。该系统提供了用户、社长、管理员三种角色，以实现社团信息的有效管理。

## 技术栈

- **架构**: B/S、MVC
- **开发语言**: Java
- **前端框架**: Vue
- **后端框架**: SpringBoot
- **数据库**: Mysql
- **持久层框架**: Mybatis
- **构建工具**: IDEA、JDK1.8、Maven

## 功能模块

### 用户模块

- 登录用户
- 注册用户
- 用户信息管理

### 超级管理员

- 所有社团管理等权限

### 社团管理员

- 本社团信息管理
- 社管活动管理
- 入团申请
- 费用管理
- 社团活动通知

### 学生用户

- 社团信息管理
- 社团活动管理
- 加入社团
- 费用管理

## 系统角色

- 用户
- 超级管理员
- 社团管理员
- 学生用户

## 运行环境

- **系统环境**: Windows
- **数据库**: Mysql8.0
